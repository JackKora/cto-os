# Personal Operating System

**Scope:** The user's own operating system as a professional — goals, how they want to show up, writing voice, personal retros, and role altitude. Cascades long-term intent down to present-week actions. Serves as the canonical source of user-level context that other modules consult via optional dependency.

**Out of scope:** Team-aggregate goals and health (Team Management); 1:1 notes and direct-report development (Managing Down, Performance & Development); external positioning (External Network & Thought Leadership); career transitions between jobs.

**Frameworks:** [OKRs — John Doerr, *Measure What Matters*](https://www.whatmatters.com/), [4Ls retrospective format](https://www.retrium.com/retrospective-techniques/the-4ls).

**Depends on:**
- Required: none (foundational)
- Optional: none

**Example tasks:**
- "Set my top 3 annual goals for this year."
- "Cascade my annual priorities into this quarter."
- "Run a personal retro on last week."
- "Update how I want to show up — I've been too reactive."
- "Add a voice sample from this Slack post."
