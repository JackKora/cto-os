# cto-os-data

This is your CTO OS data repo. It holds your personal state — module activation, 1:1 notes, goals, retros, integrations cache, and everything else the skill reads and writes.

The skill (logic, scripts, MCP server) lives separately at `~/.claude/skills/cto-os/`. The env var `CTO_OS_DATA` points at this directory.
